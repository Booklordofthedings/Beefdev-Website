beefinstall

This website explains the ways you can install the Beef programming language on your systems and also contains some further new user help.

Windows

Prebuild

Download the latest nightly or full version from the beef website. Downloading the nightly is currently recommended as the nightly contains major bugfixes that are not available in the newest release. This will also install the beef ide which is currently only available on windows.
- Newest Nightly
- 0.43.4

From source

Install the requirements listed on the beef website and follow the steps, mainly running the install script.

Linux

Prebuild

The currently most official thing you can get rn is the .deb package for BeefBuild maintained by me. The apt repository is hosted at https://beefinstall.com/Repo/. Scripts for registering this repository or even building beef or a apt repository yourself are hosted on Github.
wget -qO - https://raw.githubusercontent.com/Booklordofthedings/Beefdev-Utils/main/InstallAptFromKey.sh | bash
apt install beefbuild

Build from source

Install the requirements listed on the beef website and follow the steps, mainly running the install script. THIS WILL COMPILE LLVM, SO IT MIGHT TAKE A LONG TIME TO COMPLETE (~20min on my Ryzen 7 5700X)
